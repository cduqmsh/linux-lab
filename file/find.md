  find /usr/local/src/test -name "test*" -mtime +10

* 현재 시점에서 10일 이상된 파일 삭제

  find /usr/local/src/test -name "test*" -mtime +10 -delete

* 특정 파일 검색 및 특정 문구 검색

   find /usr \( -name "*.xml" -o -name "*.properties" -o -name "*.jar" \) -type f -exec grep -l "JMSAppender.class" {} +

   find /home/user -name "*" -type f | xargs grep "JAVA_HOME" 2>/dev/null

* 여러 개의 특정 파일 찾아서 한 파일로 병합

  find /tmp/solution/ -name "*.txt" -type f -exec cat {} >> /root/solution/info_`date +Y%-%m-%d`.txt \;
---------------------------------------------------------------------------------------------------------------------------------
* 특정 디렉토리에서 파일 검색해서 결과 파일로 떨구기

  find /usr -type f -exec ls -al {} \; >> 1.txt
